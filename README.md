# Trabalho1
Trabalho que exemplifica algoritmos de ordenação.
A cada 40 vetores, seu tamanho aumenta.
Testes são feitos com 8 tamanhos diferentes de vetores.
Os algoritmos de ordenação são:
- Insertion Sort
- Selection Sort
- Bubble Sort
- Merge Sort
- Quick Sort
- Counting Sort
- Radix Sort
- Bucket Sort
